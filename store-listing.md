## Detailed Description
```
Schelio is the Salesforce Schema Builder you always wanted — a modern, interactive ERD viewer that runs right in your browser.

CORE FEATURES

- Interactive SVG Canvas: zoom, pan, drag & drop cards, force-directed auto-layout
- Smart Field Display: fields sorted by importance with typed icons
- Relationship Lines: curved paths with arrows, color-coded (Lookup vs Master-Detail)
- Grouped Relations: multiple lookups between objects shown as one clickable line
- Cross-Object Field Search: find any field across all loaded objects instantly

DETAIL PANEL (8 TABS)

Double-click any object card to explore:
- Overview: metadata, fields with copy buttons, inline search
- Record Types: active/inactive with IDs and defaults
- Picklists: values by Record Type + dependent picklist visualization
- Layouts: page layouts, Lightning Record Pages, RT-to-Layout mapping
- Profiles: CRUD + Field-Level Security per profile with filters
- Permission Sets: CRUD permissions per Permission Set
- Validation Rules: all rules with formulas, error messages, status filters
- Automation: record-triggered Flows and Apex Triggers with events
- Health: schema health score (/100) with Salesforce limit warnings

6 EXPORT FORMATS

- PNG: high-resolution ERD screenshot
- SVG: vector export for docs
- Mermaid: ready for GitHub, Notion, Confluence
- PlantUML: UML entity diagram
- CSV: field inventory spreadsheet
- PDF Technical Spec: complete data model document with cover page, TOC, field dictionary

ADDITIONAL FEATURES

- Save & Load Layouts per org (stored locally)
- Undo / Redo (Ctrl+Z / Ctrl+Y)
- Dark & Light theme
- Context menu (right-click) on cards
- Multi-select objects (Shift+click)
- Mac keyboard shortcuts supported (Cmd)

PRIVACY & SECURITY

- Zero dependencies — no external libraries
- No data collection, analytics, or tracking
- No external server — everything stays between your browser and your Salesforce org
- Session tokens are never stored or persisted
- Open-source (MIT License): https://github.com/zmissoum/Schelio
```
